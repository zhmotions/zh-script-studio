# ZH Script Studio

A ZH Motions extension for **Adobe Premiere Pro** & **After Effects** that turns your
script files into **editable, animated** titles and subtitles — straight on the timeline.

## ✨ Features
- 📄 Open **.docx / .txt / .srt** scripts (auto-reloads your last file)
- ➕ **Title** — selected text → editable, animated title at the playhead
- ≣ **Batch** — every line → its own sequential title
- 💬 **Subtitles** — full script → timecoded `.srt` (Premiere) or timed text layers (After Effects)
- 📋 **Paste** — clipboard text → timeline title
- ⚑ **Marker**, ⧉ **Copy**, 🎨 **Mark text** (colour/highlight)
- 🎬 **Custom templates** — your own `.mogrt` (Premiere) or `.ffx` preset (After Effects), saved permanently
- 🅰️ **Auto-installs bundled fonts** (Bengali + English) — no manual setup
- ✏️ In-panel **edit** (fix typos) + system-font picker

## 🌏 Bengali + English
Bundled free Google fonts (Hind Siliguri, Noto Sans Bengali, Anek Bangla, Inter, Poppins)
auto-install on first launch, so Bengali and English render everywhere.

## 💻 Requirements
- Premiere Pro / After Effects **22.0 (2022) or newer**
- macOS or Windows

## 📥 Install
1. Download **ZHScriptStudio-v2.0.zxp** from [Releases](../../releases).
2. Install with any ZXP installer (e.g. *ZXP/UXP Installer*).
3. Restart Premiere Pro / After Effects.
4. Open **Window → Extensions → ZH Script Studio**.

> Self-signed build — your installer may show a one-time trust prompt. This is normal.

## 🔑 Activation
ZH Script Studio requires a license key. Enter your key on first launch.
